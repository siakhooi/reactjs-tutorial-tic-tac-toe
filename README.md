## setup

```bash
npx create-react-app reactjs-tutorial-tic-tac-toe
cd reactjs-tutorial-tic-tac-toe
cd src
rm -f *   # del * (windows)
cd ..
```

### add gh-pages

```bash
npm install --save-dev gh-pages react-test-renderer
```
